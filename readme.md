## Tech Stack

**Client:** Pure React native components


## Run Locally

Clone the project

`
  git clone https://github.com/LuthfanM/Map-Lockation
`

Install MapLock with pnpm

`
  npm install
`

And run the project via

`
  npx expo start
`
